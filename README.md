# Møbel

*(czyt. Mebel)*

## Uruchamianie aplikacji

Należy zweryfikować URL do API w pliku `src/apiUrls.json`

Uruchamianie wersji developerskiej:

    $ npm run start

Budowanie wersji produkcyjnej:

    $ npm run build

Zbudowana wersja produkcyjna jest dostępna w folderze `build`.
